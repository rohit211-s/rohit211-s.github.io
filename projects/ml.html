Sentiment Analysis of 2023 Nigerian Presidential Election Tweets Using PowerBI

  • Github URL: Project Link
  • Tools Used:

    Power BI, Power Query, Azure Cognitive Services (Text Analytics), Kaggle

In this project, I explored the public sentiment surrounding the 2023 Nigerian Presidential Election by analyzing Twitter data to gauge the mood and opinions of Nigerians in the election's aftermath. Using Power BI and Azure Cognitive Services, I processed thousands of tweets to capture real-time reactions, categorize sentiments, and highlight key topics and terms central to this digital conversation.

The dataset, sourced from Kaggle, included rich details such as tweet content, user information, and tweet engagement, providing a comprehensive foundation for sentiment analysis. I cleaned and transformed the data in Power Query, structuring it to focus on meaningful insights. A custom sentiment score was then applied to each tweet, labeling sentiments as Positive, Neutral, or Negative to make the findings intuitive and actionable.

Through Power BI visualizations, the analysis revealed intriguing patterns: while Negative sentiments had a notable presence, Positive sentiments took a leading role in the overall sentiment landscape, showing a more favorable or hopeful perspective from the public. Key phrases like "INEC," "BVAS," and "Election" emerged as focal points in discussions, reflecting the issues that mattered most to users.

The project culminated in a detailed, visually engaging report that not only shed light on the sentiment trends of Nigerian Twitter users during this pivotal period but also underscored the dynamic, nuanced nature of online discourse. This analysis brings a neutral, data-driven lens to an emotionally charged topic, offering valuable insights into public sentiment trends within Nigeria's political sphere.
