Accelerating MNIST Training with Custom CUDA Kernels

  • Github URL: Project Link
  • Tools Used:

    CUDA, PyTorch, Numpy, Pandas, Matplotlib

This project dives into the potential of low-level CUDA programming to enhance the training speed of neural networks. With a strong background in Python-based frameworks like PyTorch and JAX, I sought to bridge the gap between high-level convenience and low-level performance by directly implementing CUDA kernels. By crafting custom CUDA solutions, I explored and addressed performance bottlenecks present in widely-used frameworks.

The results were eye-opening. PyTorch, even when tuned for maximum efficiency with PyTorch 2.0's torch.compile, lagged significantly behind. My CUDA implementation outperformed PyTorch, especially in smaller networks, achieving up to 6x speed improvements. Notable optimizations included using fp16 precision to double throughput in matrix multiplications, and carefully managing data preloading to reduce transfer delays. This project also honed my skills with CUDA, challenging me to balance speed with code readability to make the project accessible for others interested in CUDA.
